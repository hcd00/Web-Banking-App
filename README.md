# Web-Banking-App
Web Banking and ATM simulation

## JDK 21
Our project is based on JDK version 21. The downnload link is below. 
https://www.oracle.com/java/technologies/downloads/#java21

After confirming you have the compatible Java installed you can compile this program in the terminal using 

```
javac main.java
java main
```

or you can use your native IDE provided you have the necessary plug-ins and select "run".
